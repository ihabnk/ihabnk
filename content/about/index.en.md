## Welcome

This demo highlights how the Blowfish theme looks with a bilingual setup. The site is configured for Arabic as the default language and English as the alternate, using the standard configuration files from the Blowfish documentation.

## What’s Included

- Multilingual content powered by Hugo’s language features.
- Card-style blog listing with thumbnails and blurred hero headers inside each article.
- Adjustable theme parameters in `config/_default/params.toml`.
