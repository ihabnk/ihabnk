## Getting Started

Customize this page in `content/_index.en.md` and update the site configuration in `hugo.toml` to match your project details.

- Remember to create posts inside `content/blog/`.
- Use `hugo server` for previews and `hugo server -D` to include draft content.
- Explore the theme docs at [blowfish.page/docs](https://blowfish.page/docs/) for layout options and components.
